1. 기본적인 설치 방법

1-1. 연동 Flow

이니페이모바일™ 은 복잡한 카드사와의 연계를 당사에서 처리하고, 가맹점에는 통일화된 규격을 제시함에 따라, 보다 편리하게 모바일 결제시스템을 구축할 수 있게 합니다.

"인증/승인 분리 방식 (이하 2 Transaction 이라 명명)" 을 기본 Flow 로 하며, 일부 지불수단에 한하여, "인증/승인통합 방식 (이하 1 Transaction 이라 명명)" 을 사용합니다.




2. 결제요청 페이지 작성

요청 URL

상점 페이지에서 이니페이모바일TM 접속 시, 지불수단 및 통신방식 별로 상이한 URL 을 사용 합니다. 이에, 하기의 URL 을 참고하시기 바랍니다

HOST
지불수단별 상세 URL
기본 mobile.inicis.com TLS 1.2 지원 mobile.inicis.com:8443

신용카드 https://{HOST}/smart/wcard/

계좌이체 https://{HOST}/smart/bank/

가상계좌 https://{HOST}/smart/vbank/

휴대폰 https://{HOST}/smart/mobile/

문화상품권 https://{HOST}/smart/culture/

해피머니상품권 https://{HOST}/smart/hpmn/

스마트문상 https://{HOST}/smart/dgcl/

ex) 신용카드 연동 시 : https://mobile.inicis.com/smart/wcard/

신용카드 연동 시(TLS 1.2 지원) : https://mobile.inicis.com:8443/smart/wcard/
